# Fleet Control Dashboard

Dashboard ديناميكي لمراقبة السائقين، مقارنة الرحلات، وكشف الشذوذ من ملفات Excel أو CSV.

## Features

- Upload CSV or Excel files
- Arabic and English column detection
- Executive dashboard
- Driver comparison
- Route intelligence
- Suspicious trip detection
- Download filtered or flagged data

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
